# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `GET /api/users`
- `GET /api/users/:id`
- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Campaigns

- `GET /api/campaigns/`
- `GET /api/campaigns/:id`
- `POST /api/campaigns`
- `PATCH /api/campaigns/:id`
- `DELETE /api/campaigns/:id`

### Rewards

- `GET /api/rewards`
- `POST /api/rewards`
- `PATCH /api/rewards/:id`
- `DELETE /api/rewards/:id`


### Contributions

- `GET /api/contributions`
- `POST /api/contributions`

### Categories

- `GET /api/categories`

### Followings

- `GET /api/followings`
- `POST /api/followings`
- `DELETE /api/followings`
